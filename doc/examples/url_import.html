Example: Change URL of an importing library.

For using OGRA in offline mode you have to download all required graphical libraries of third parties and import them, before initialization of OGRA. OGRA is checking namespace for required libraries during it's initialization. If required names are not defined, OGRA will try to download all required libraries from preset URLs. You can also overwrite these preset URLs.

HTML:

Import OGRA JS and create element for future chart.


	<script type="text/javascript" src="https://raw.github.com/CZNIC-Labs/ogra/master/ogra.js"></script>

	<div id="my_chart"></div>

Javascript:

Prepare your data and create chart.


	var data = {"cols":[{"label":"Year", "type":"string"},{"label":"Sales", "type":"number"},{"label":"Expenses", "type":"number"}],"rows":[{"c":[{"v":"2004"},{"v":1000},{"v":400}]},{"c":[{"v":"2005"},{"v":1170},{"v":460}]},{"c":[{"v":"2006"},{"v":660},{"v":1120}]},{"c":[{"v":"2007"},{"v":1030},{"v":540}]}]};			
	
	// We want to use different version of jQuery.
	Ogra.URL_JQUERY = "http://code.jquery.com/jquery-1.9.0.min.js";

	/* Preset URLs
	Ogra.URL_JQUERY ="//ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js";
	Ogra.URL_GOOGLE = "//www.google.com/jsapi";
	Ogra.URL_DYGRAPHS = "//dygraphs.com/dygraph-combined.js";
	Ogra.URL_HIGH = "http://code.highcharts.com/highcharts.js";
	Ogra.URL_FLOT = "//raw.github.com/paradoxxxzero/flot/28f2377382b3af97c82cb3ebc081140b49fa9579/jquery.flot.js";
	Ogra.URL_FLOT_PIE = "//raw.github.com/flot/flot/master/jquery.flot.pie.js";
	*/

	Ogra.graph("my_chart", data, "column", "high", {} );
