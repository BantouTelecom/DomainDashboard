About OGRA

OGRA is a javascript library that allows display of online charts using several third party libraries using one unified format.

For better understanding you can jump directly to the examples.

Supported browsers

  • Chrome
  • Firefox
  • Internet Explorer 6+
  • Safari
  • Opera

Most of the compatibility is based on jQuery v 1.7.1. Please visit jQuery Current Active Support for more details about supported web browsers.
